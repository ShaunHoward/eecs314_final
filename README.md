# eecs314_final
An image filter program for .bmp images.

There are several types of filters, including:

Saturation

Hue

Brightness

ShadowFill

Inversion

Grayscale

*Edge detection 

This was a final group project for EECS314, Spring 2015 at Case Western Reserve University.
